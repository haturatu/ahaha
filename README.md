ahaha - Ansi HTML Adapter
=======================

Converts ANSI escape sequences of a unix terminal to HTML code, and can also
convert HTML generated by ahaha back to ANSI text.

![ahaha screenshot](/screenshot.png?raw=true "ahaha screenshot")

Examples
========

* `ahaha --help | ahaha --black --title "the awesome ahaha help"> ahaha-help.htm`
  * Creates an HTML file of the help of ahaha with black background
* `colordiff oldfile.c newfile.c | ahaha > colordiff.htm`
  * Creates an HTML file with a colorful diff-output of two files "oldfile.c" and "newfile.c" with white background
* `ls --color=always | ahaha --pink > ls.htm`
 * Creates an HTML file with a colorful ls-output with pink background.
* `echo q | htop | ahaha --black --line-fix > htop.htm`
  * Creates an HTML file with the output of htop. You have to use --line-fix due the other new-line-commands htop uses.
* `MAN_KEEP_FORMATTING=1 COLUMNS=80 man ahaha | ul | ahaha > man-ahaha.htm`
  * Creates an HTML file with the man page of ahaha. Man uses nroff's bold and underline, which ul converts to SGR.
* `ahaha --reverse -f output.html > restored.txt`
  * Converts HTML generated by ahaha back to ANSI text.

Differences From Upstream
=========================

This repository is a fork of `theZiz/aha`.

* The installed command name is `ahaha` instead of `aha`.
* `ahaha --reverse` converts HTML generated by the tool back into ANSI text.
* Generated HTML carries fork metadata pointing to this repository and the upstream project.
* Original author: Alexander Matthes (`ziz@mailbox.org`). Fork maintainer: haturatu (`taro@eyes4you.org`).

Compilation / Installation
==========================

## Build From Source

ahaha has no dependencies except for a C compiler and `make`.

Clone this repo `git clone git@github.com:haturatu/ahaha.git`.
Move into the repo directory `cd ahaha`.
To compile just type `make`.
To install ahaha to `/usr/local/` type `make install`.
You can change the installation directory with `make install PREFIX=/your/path`.
You can override the man directory with `make install MANDIR=/your/path/man`
